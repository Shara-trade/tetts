# Руководство по админ-панели Lazy Farmer Bot

## Доступ
Только создатель бота (ID из переменной `CREATOR_ID` в `.env`)

## Команды

### Главная
- `/admin` - Открыть админ-панель

### Растения
- `/addplant код название икона цена_покупки цена_продажи время_роста` - Добавить новое растение

## Разделы админ-панели

### 1. Игроки (👥)
- Просмотр статистики пользователей
- Поиск по ID/username
- Выдача монет
- Бан/разбан

### 2. Растения (🌱)
- Список всех растений
- Редактирование цен, времени роста
- Включение/выключение
- Добавление новых

### 3. Экономика (💰)
- Настройка цен
- Настройка множителей
- Баланс экономики

### 4. Промокоды (🎁)
- Создание промокодов
- Просмотр активных
- Отключение

### 5. Ачивки (🏆)
- Настройка достижений
- Добавление новых
- Изменение наград

### 6. Квесты (📜)
- Ежедневные задания
- Еженедельные задания
- Настройка наград

### 7. Фермеры (👤)
- Типы фермеров
- Цены найма
- Бонусы

### 8. Рассылка (📢)
- Все игроки
- Активные сегодня
- Новички

### 9. Статистика (📊)
- DAU/WAU
- Экономика
- Активность

### 10. Логи (📋)
- Действия админов
- История изменений

## Примеры использования

### Добавление растения
```
/addplant golden_wheat Золотая_пшеница 🌾 1000 2500 600
```

### Выдача монет игроку
Через меню "Игроки" → "Найти игрока" → "Выдать монеты"

### Создание промокода
Через меню "Промо" → "Создать промокод"

## Безопасность
- Все действия логируются
- Только создатель имеет доступ
- Историю действий можно посмотреть в разделе "Логи"
