# Админ-панель Lazy Farmer v2.0

## Обзор

Полностью переработанная админ-панель с инлайн-интерфейсом. Единственная команда для входа — `/admin`.

## Ключевые особенности

### 🔐 Система ролей
| Роль | Эмодзи | Права |
|------|--------|-------|
| Creator | 👑 | Полный доступ |
| Admin | ⚡️ | Игроки, промо, бонусы, экономика, ачивки |
| Moderator | 🛡️ | Поиск игроков, временный бан (до 24ч), просмотр профилей |

### 📁 Структура меню

#### Главное меню (адаптивное)
**Для всех ролей:**
- 👥 Игроки
- 🌱 Растения
- 💰 Экономика
- 📢 Рассылка
- ❓ Помощь

**Дополнительно для Admin и Creator:**
- 🎁 Промо-акции
- ⭐️ Ежедневный бонус
- 🏆 Управление ачивками

**Только для Creator:**
- 👑 Управление админами
- 📊 Логи действий
- ⚙️ Настройки системы

## Разделы

### 👥 Игроки
- 🔍 Поиск по ID
- 🔍 Поиск по @username
- 📋 Последние 10 игроков
- 💰 Топ по балансу
- 🏆 Топ по уровню
- 📊 Онлайн статистика

**Профиль игрока:**
- Полная информация о пользователе
- Ресурсы и инвентарь
- Статистика активности
- Кнопки управления

**Действия с игроком:**
- 💰 Выдать монеты
- 💎 Выдать кристаллы
- 🌱 Выдать предмет
- 🔨 Забрать ресурсы
- ⛔️ Забанить / ✅ Разбанить
- 📜 История действий
- 🏅 Ачивки игрока
- 💬 Отправить сообщение
- ⚠️ Сбросить прогресс (Creator)
- 🗑 Удалить аккаунт (Creator)

### 🌱 Растения (Admin, Creator)
Пошаговый мастер создания:
1. 🆔 ID растения
2. 🏷️ Название
3. 🎨 Эмодзи
4. ⏱️ Время роста
5. 💰 Цена семян
6. 💵 Цена продажи
7. 🌾 Урожайность
8. ⭐ Требуемый уровень
9. ✨ Опыт за сбор
10. ✅ Подтверждение

### 🎁 Промо-акции (Admin, Creator)
Пошаговый мастер создания:
1. 📝 Код промо
2. 📋 Тип (по времени / по количеству)
3. 🎁 Тип награды
4. 💰 Количество награды
5. 📊 Лимиты
6. ✅ Подтверждение

### 📢 Рассылка (все роли)
- Ввод контента (текст/HTML)
- Выбор аудитории:
  - 👥 Все игроки
  - 📅 Активные сегодня
  - 🆕 Новички (до 7 дней)
- Предпросмотр
- Тестовая отправка
- Фоновая рассылка

### 👑 Управление админами (Creator)
- 📋 Список текущих админов
- ➕ Назначить админа
- ➕ Назначить модератора
- 🗑 Снять роль

### 📊 Логи (Creator)
- 📅 Статистика по группам
- 👑 Админ-действия
- 💰 Экономика
- 🌱 Прогресс игроков
- 🛡️ Безопасность
- ⚙️ Очистка старых логов

## Навигация

### Стандартные кнопки
- 🔙 Назад — возврат на предыдущий уровень
- 🏠 Главное меню — возврат в корень админки
- ❌ Отмена — выход из FSM

### Пагинация
Для всех списков используется:
```
◀️ [1/5] ▶️
```

## FSM (Машина состояний)

Все сложные операции используют FSM для пошагового ввода:
- Выдача ресурсов
- Создание растений
- Создание промокодов
- Рассылка
- Управление админами

## Логирование

Все действия админов логируются:
- Кто выполнил действие
- Какое действие
- На кого направлено
- Детали и причины
- Время действия

## Безопасность

- Модераторы не могут банить админов
- Модераторы ограничены баном до 24 часов
- Creator не может быть снят с роли
- Все действия логируются

## Технические детали

### Файлы
- `admin_panel_full.py` — основной файл админ-панели
- `states.py` — FSM состояния (добавлены AdminFSM)
- `database.py` — методы для работы с БД

### Подключение в main.py
```python
from admin_panel_full import router as admin_router
```

### Требования
- aiogram 3.x
- FSMContext для пошаговых операций
- InlineKeyboardMarkup для интерфейса
