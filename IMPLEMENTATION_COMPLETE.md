## 🚀 ЗАПУСК

### Требования:
- Python 3.10+
- aiogram 3.x
- aiosqlite

### Установка:
```bash
pip install aiogram aiosqlite
```

### Настройка:
Создать файл `.env`:
```
BOT_TOKEN=your_token_here
CREATOR_ID=your_telegram_id
BOT_USERNAME=YourBotName
OFFICIAL_CHAT_LINK=https://t.me/your_chat
OFFICIAL_CHAT_USERNAME=your_chat
```

### Запуск:
```bash
cd admin
python main.py
```
