# Обновление Lazy Farmer Bot

## Добавленные функции

### 1. Система квестов (📜 Квесты)
- Ежедневные задания с прогресс-барами
- Типы квестов: сбор урожая, посадка, заработок монет, вход в игру
- Автоматическое обновление прогресса
- Награды: монеты и предметы

### 2. Система достижений (🏆 Достижения)
- Достижения: Первый урожай, Сборщик, Фермер месяца, Богач и др.
- Награды за достижения: монеты + множители дохода
- Автоматическая проверка при действиях
- Уведомления о новых достижениях

### 3. Сезонные события
- События с увеличенными наградами
- Уникальные сезонные семена
- Лидерборд событий
- Обратный отсчет времени

### 4. Улучшенная админ-панель
#### Роли:
- **👑 Создатель** (ID: 7852152665) - полный доступ
- **⚡ Администратор** (ID: 701414064) - выдача ресурсов, баны, назначение модераторов
- **🛡️ Модератор** - просмотр профилей, временные баны (до 24ч)

#### Команды админов:
- `/givecoins @username сумма [причина]` - Выдать монеты
- `/giveitem @username предмет количество [причина]` - Выдать предмет
- `/ban @username причина [часы]` - Забанить (перма/временно)
- `/tempban @username причина часы` - Временный бан (для модераторов)
- `/unban @username` - Разбанить
- `/finduser @username` - Профиль игрока
- `/admins` - Список админов (только создатель)
- `/addadmin @username роль` - Назначить админа
- `/addmoderator @username` - Назначить модератора
- `/removeadmin @username` - Снять роль
- `/logs` - Логи действий (только создатель)

### 5. ASCII-графика и прогресс-бары
- Визуальное отображение растений на разных стадиях
- Прогресс-бар роста в реальном времени
- Анимированные эмодзи

### 6. Система уведомлений
- Автоматические уведомления о созревшем урожае
- Напоминания о ежедневном бонусе (12+ часов)
- Уведомления о завершении квестов
- Фоновая проверка каждые 5 минут

### 7. Новые кнопки меню
- 📜 Квесты - ежедневные задания
- 🏆 Достижения - прогресс и награды

## Как остановить бота

1. **Ctrl+C** в терминале
2. **PowerShell:**
   ```powershell
   Get-Process python | Stop-Process -Force
   ```
3. **Task Manager:** Найти python.exe и закрыть

## Тестирование

Запусти бота:
```bash
python main.py
```

Проверь:
1. Нажми /start
2. Проверь кнопки 📜 Квесты и 🏆 Достижения
3. Посади и собери урожай - должны обновляться квесты
4. Проверь /admin с разными ролями
5. Проверь уведомления (подожди 5 мин или посмотри в логи)
