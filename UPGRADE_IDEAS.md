## Как остановить бота:

### Способ 1: Ctrl+C (если в терминале)
Просто нажми `Ctrl+C` в окне терминала где запущен бот.

### Способ 2: Task Manager
1. Нажми `Ctrl+Shift+Esc`
2. Найди процесс `python.exe`
3. Выбери и нажми "Снять задачу"

### Способ 3: Командная строка
```cmd
taskkill /F /IM python.exe
```

### Способ 4: PowerShell
```powershell
Get-Process python | Stop-Process -Force
```

### Способ 5: Если в VS Code
Нажми на квадратик (Stop) в верхней панели рядом с терминалом.
