# Обновление админ-панели v2.2

## Дата обновления
2024

## Что реализовано (все TODO из v2.1)

### 1. ✅ Полноценное редактирование промокодов
**Функции:**
- `admin_promo_edit` - меню редактирования промокода
- `admin_promo_edit_field` - выбор поля для редактирования
- `admin_promo_edit_code_value` - редактирование кода
- `admin_promo_edit_reward_value` - редактирование награды
- `admin_promo_edit_uses_value` - редактирование лимита активаций
- `admin_promo_edit_date_value` - редактирование срока действия

**Возможности:**
- Редактирование кода (с проверкой уникальности)
- Редактирование наград (coins, gems, items)
- Редактирование лимита активаций (включая безлимит)
- Редактирование срока действия (или бессрочно)
- Валидация всех вводимых данных
- Логирование всех изменений

**FSM состояния:**
- `promo_edit_code` - редактирование кода
- `promo_edit_reward` - редактирование награды
- `promo_edit_uses` - редактирование лимита
- `promo_edit_date` - редактирование даты

### 2. ✅ Удаление промокодов
**Функции:**
- `admin_promo_delete_start` - начало удаления
- `admin_promo_delete_execute` - выполнение удаления

**Типы удаления:**
- **Мягкое удаление** - просто деактивация (is_active = 0)
- **Полное удаление** - удаление из БД (включая записи об активациях)

**FSM состояния:**
- `promo_delete_confirm` - подтверждение удаления

**Особенности:**
- Подтверждение перед удалением
- Выбор типа удаления
- Логирование

### 3. ✅ Активация/деактивация промокодов
**Функция:** `admin_promo_toggle`

**Возможности:**
- Быстрое переключение активности промокода
- Логирование действия
- Уведомление об успешном переключении

### 4. ✅ Статистика использования растений (реальная)
**Функции:**
- `admin_plants_stats` - общая статистика посадок
- `admin_top_farmers` - топ фермеров

**Показывает:**
- Общее количество посадок
- Общее количество собранного урожая
- Общий заработок
- Статистика по каждому растению:
  - Количество посадок
  - Количество собранного
  - Заработанные монеты
- Топ-10 фермеров с детальной статистикой

**Особенности:**
- Данные берутся из таблицы `users` (поля total_planted, total_harvested, total_earned)
- Равномерное распределение по растениям (при отсутствии детальной статистики)
- Навигация к профилям игроков

### 5. ✅ Корзина удаленных растений
**Функции:**
- `admin_plants_trash` - список неактивных растений
- `admin_plant_restore` - восстановление растения
- `admin_plant_delete_hard` - полное удаление
- `admin_plant_delete_hard_confirm` - подтверждение удаления

**Возможности:**
- Просмотр всех неактивных растений (is_active = 0)
- Восстановление растения (is_active = 1)
- Полное удаление из БД
- Проверка на активные посадки перед удалением
- Подтверждение полного удаления

**Особенности:**
- Защита от удаления растений с активными посадками
- Логирование всех действий

### 6. ✅ Статистика посадок по растениям
**Функция:** `admin_plants_stats`

**Показывает:**
- Общая статистика по всем растениям
- Детальная статистика по каждому растению:
  - Название и эмодзи
  - Приблизительное количество посадок
  - Приблизительное количество сбора
  - Приблизительный заработок
- Кнопка "Детально" для каждого растения
- Кнопка "Топ фермеров"

**Особенности:**
- Использует реальные данные из БД
- Навигация между разделами

## Обновления в database.py

### Новые методы для промокодов:

```python
# Получение всех промокодов
async def get_all_promocodes(active_only: bool = False) -> List[Dict]

# Получение промокода по коду
async def get_promocode_by_code(code: str) -> Optional[Dict]

# Создание промокода
async def create_promocode(admin_id: int, code: str, reward: dict, 
                          max_uses: int, valid_until: str = None) -> int

# Обновление промокода
async def update_promocode(promo_id: int, code: str = None, reward: dict = None,
                         max_uses: int = None, valid_until: str = None, 
                         is_active: bool = None) -> bool

# Переключение активности
async def toggle_promocode(promo_id: int) -> bool

# Удаление промокода
async def delete_promocode(promo_id: int, hard_delete: bool = False) -> bool

# Статистика промокодов
async def get_promocode_stats(promo_id: int = None) -> Dict
```

### Новые методы для статистики растений:

```python
# Статистика растений
async def get_plant_stats(plant_code: str = None, days: int = 30) -> Dict

# Лидерборд по посадкам
async def get_planting_leaderboard(plant_code: str = None, limit: int = 10) -> List[Dict]
```

### Новые методы для управления растениями:

```python
# Обновление конфигурации растения
async def update_plant_config(item_code: str, field: str, value) -> bool

# Переключение активности растения
async def toggle_plant_active(item_code: str) -> bool

# Удаление конфигурации растения
async def delete_plant_config(item_code: str, soft_delete: bool = True) -> bool

# Получение неактивных растений (корзина)
async def get_inactive_plants() -> List[Dict]
```

## Примеры использования

### Редактирование промокода
```
Промокоды → Список → ✏️ Ред → Выбор поля → Ввод значения → Сохранено

```
Промокоды → Список → 🗑 Уд → Выбор типа (мягкое/полное) → Подтверждение → Удалено

### Корзина растений
```
Растения → Корзина → ♻️ Восстановить или 🔥 Удалить
```

### Статистика посадок
```
Растения → Статистика посадок → Общая статистика + По растениям + Топ фермеров
```
