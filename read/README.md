# Lazy Farmer Bot

Телеграм-бот фермерской игры с системой грядок, магазином, престижем и промокодами.

## Функции

- 4 грядки для посадки культур
- Магазин с семенами, удобрениями и улучшениями
- Система ежедневных бонусов
- Инвентарь (амбар)
- Система престижа
- Промокоды
- Админ-панель

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
```bash
pip install -r requirements.txt
```
3. Создайте файл `.env` и добавьте токен бота:
```
BOT_TOKEN=your_bot_token_here
```
4. Запустите бота:
```bash
python main.py
```

## Команды

- `/start` - Начать игру / Главное меню
- `/help` - Помощь
- `/admin` - Админ-панель (только для админов)

## Структура проекта

- `main.py` - Точка входа, инициализация бота
- `handlers.py` - Обработчики команд и сообщений
- `database.py` - Работа с базой данных SQLite
- `keyboards.py` - Клавиатуры и кнопки
- `states.py` - Состояния FSM
- `admin.py` - Админ-функции
- `init_db.sql` - Схема базы данных и начальные данные

## Требования

- Python 3.8+
- aiogram 3.x
- aiosqlite
- python-dotenv
